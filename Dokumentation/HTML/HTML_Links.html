HTML Links und Images

Links

In HTML starten Links mit dem Start-Tag <a>. In diesem Start-Tag muss aber noch mindestens ein Attribut sein. Das href-Attribut. Dieses bstimmt die URL des Links. Da in HTML alle Links Hyperlinks sind, kann das auch ein Dateipfad sein. Nach dem Start-Tag folgt der Text der angezeigt werden soll. Der Link endet mit einem End-Tag </a>. So sollte ein Link mindestens aussehen:
<a href="https://youtube.com">Hier kannst du Videos schauen.</a>
Hier kannst du Videos schauen

Weitere Attribute für Links

target

Target sagt, wie bzw. wo der Link geöffnet wird.
_blank: In einem neuen Tab oder Fenster.
_self: Im selben Tab oder Fenster. (Standart)

title

Der Title wird als Tooltip angezeigt wenn man über den Link fährt.
Er kann irgendein Text sein.

mailto

Mit mailto kann man einen Link zu einer E-Mail-Adresse machen. Ich empfehle dabei, den target auf _blank zu setzen. Er gehört aber zu href.
So sieht ein Link zu einer E-Mail-Adrese aus:
<a href="mailto:silas.hardegger@tkb.ch" target="_blank">Email senden</a>
Email senden

Images

Images werden mit dem Start-Tag <img> eingeleitet. Auch Images brauchen Attribute im Start-Tag. Hier gibt es zwei relevante Attribute:

src

Das src Attribut zeigt dem Skript, wo sich das Bild befindet. Dabei gibt man einen Dateipfad an.

alt

Mit dem alt Attribut wird ein alternativer text angegeben, der angezeigt wird, wenn das Bild nicht verfügbar ist. Falls das alt Attribut zum Einsatz kommt:
Sollte man nachschauen, ob der Dateipfad stimmt. Sollte man die Seite neuladen. Sollte man sichergehen, dass die Bilddatei nicht beschädigt ist.

Weitere Atribute für Images

heigth

Das heigth Attribut bestimmt die Höhe des Bildes. Es wird in Pixeln (px) angegeben.

width

Das width Attribut bestimmt die Breite des Bildes. Es wird ebenfalls in Pixeln (px) angegeben.
