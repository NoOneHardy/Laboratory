HTML Elements

Headings

Die Headings sind sogenannte Ueberschriften. Es gibt 6 verscheidene Varianten - h1 bis h6. H1 ist die Grösste und h6 ist die Kleinste. Sie haben wie alle anderen Elemente einen Start- und einen End-Tag.
Beispiel:
<h1>Ich bin eine grosse Überschrift</h1>
<h6>Ich bin eine kleine Überschrift</h6>

Ich bin eine grosse Überschrift

Ich bin eine kleine Überschrift

Paragraphs

Paragaphs sind unwichtiger als Headings. Darum werden sie auch kleiner geschrieben. Sie ignorieren die Darstellung, wie der Text im Skript geschrieben ist. Beim folgenden Satz zum Beispiel werden die Leerzeichen ignoriert: 
<p>Dieser Satz wird ganz           normal dargestellt.</p>
Dieser Satz wird ganz normal dargestellt.

Preformatted Text

Weil Paragraphs keinen Text formatiert ausgeben, gibt es ein Element, welches den Text genauso ausgibt wie er im Skript steht. Der Preformatted Text hat einen Start- und End-Tag.
<pre>Hallo            Welt</pre>
Diese Texte können auch mehrere Zeilen lang sein:
<pre>
Hallo
Ich bin ein formatierter Text.
Ich bin mehrere Zeilen lang.
Und Abstände        werden eingehalten.
</pre>
				

Hallo
Ich bin ein formatierter Text.
Ich bin mehrere Zeilen lang.
Und Abstände        werden eingehalten.

Line Breaks

Um einen Zeilenumschlag zu machen, braucht man sogenannte Line Breaks. Line Breaks haben keinen End-Tag. Nach einer Zeile, schreibt man einfach <br>. Mit einem Line Break kann man sich auch einige Tags sparen. Du musst zum Beispiel nur einen Start- und einen End-Tag machen, wenn du nach jeder Zeile ein <br> setzt.

Horizontal Rules

Horiontal Rules sind Trennlinien. Man verwendet sie um Texte voneinander zu trennen. Horizontal Rules werden ähnlich gemacht, wie Line Breaks. Sie haben nur den Start-Tag. <hr>
