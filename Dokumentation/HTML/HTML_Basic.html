HTML Basic

Tags

Jedes Element in HTML ist folgendermassen aufgebaut:
<Start-Tag>Inhalt<End-Tag>.

!DOCTYPE html

Jedes HTML-Skript startet mit <!DOCTYPE html>. So wird definiert, dass das Dokument in HTML geschrieben ist. Der <!DOCTYPE html>-Tag braucht KEINEN End-Tag.

Html

Der <html>-Tag wird wird als zweitens geschrieben. Er beendet das Dokument auch mit seinem End-Tag <html>.

Title

Mit dem <title>-Tag kann man den Namen der Website ändern. Mit "Name" wird der Text oben im Browser-Tab gemeint. Auch dieser Tag endet mit seinem End-Tag <title>.
Beispiel:
<!DOCTYPE html>
<html>

<title>Name der Website</title>

</html>

Body und Head

Ein HTML-Dokument hat einen Körper. Dieser wird mit dem <body>-Tag eingeleitet und mit </body> beendet. Alles was zwischen dem Start- und dem End-Tag steht, wird angezeigt. Zusätzlich gibt es noch den <head>-Tag, welcher auch mit </head> aufhört. Er bildet die Kopfzeile der Website.
Beispiel:
<!DOCTYPE html>
<html>

<title>Name der Website</title>

<body>

</body>
</html>
