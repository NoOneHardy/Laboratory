JavaScript Output

innerHTML

Um den Inhalt eines HTML Elements mit einem JavaScript-Programm zu ändern, gibt es die
innerHTML-Property. Im folgenden Beispiel wird der Inhalt des Paragrapen mit der ID para1 verändert.
<!DOCTYPE html>
<html>
<body>

<p id="para1">Ich bin ein Paragraph</p>

<script>
document.getElementById("para1").innerHTML =
"Der script-Tag hat mich verändert."
</script>

</body>
</html>

Ich bin ein Paragraph

document.write()

Man kann mit document.write() beim Laden der Seite per JavaScript Text hinzufügen. Wenn man allerdings document.write() braucht nachdem die Seite geladen hat, wird der ganze Inhalt der Seite gelöscht.
Beispiel:
<!DOCTYPE html>
<html>
<body>

<script>
document.write(5 + 6);
</script>

</body>
</html>

window.alert()

Mit window.alert() kann man über eine Pop-Up Meldung ein Nachicht anzgeigen lassen. Man kann das window. auch weglassen. alert() bewirkt genau dasselbe.
Beispiel:
<!DOCTYPE html>
<html>
<body>

<script>
alert("Schön dich zu sehen.");
</script>

</body>
</html>

console.log()

console.log() ist für den normalen Benutzer der Seite nicht sichtbar. Dieser Output wird nur sichtbar, wenn man einen Rechtsklick macht und die Option untersuchen wählt. Die Nachricht wird im Bereich Consoleangezeigt. Das wird man wahrscheinlich nur selten brauchen.
Beispiel:
<!DOCTYPE html>
<html>
<body>

<script>
console.log("Hallo");
</script>

</body>
</html>

window.print()

Mit window.print() kann man sich die ganze Seite ausdrucken lassen. Hier gilt wieder das gleiche wie bei window.alert(). Man kann auch hier nur print() eingeben und es druckt trotzdem.
Beispiel:
<!DOCTYPE html>
<html>
<body>

<script>
print();
</script>

</body>
</html>
